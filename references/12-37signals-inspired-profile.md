# 37signals Inspired Profile

Use this profile only when the user asks for 37signals-style conventions.

## Core Tendencies

- Prefer vanilla Rails over additional abstraction layers.
- Prefer rich domain models with thin controllers.
- Prefer resourceful routing and noun-based state resources.
- Prefer small cohesive concerns for horizontal behavior reuse.
- Prefer pragmatic iteration over premature architecture.

## Cautions

- Treat this profile as inspiration, not strict doctrine.
- Validate patterns against your app constraints and team practices.
- Avoid introducing churn by forcing style migrations in feature PRs.
